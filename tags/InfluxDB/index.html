InfluxDB 内存优化
2020年05月08日 937 字 大概 4 分钟 Docker Docker InfluxDB Linux

最近给小程序的后端日志加上了实时输出到前端功能，为了展示就得保存日志，又考虑到日志是时序数据，自然就选择了时序数据库 InfluxDB，并且通过 Docker 部署。刚上线的一两天没有什么问题，各方面运行都很平稳，没过几天的一个早上突然收到了阿里云的报警，说内存占用超过了 95%。

这便是噩梦的开始。
